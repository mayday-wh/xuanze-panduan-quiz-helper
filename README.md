# 选择判断刷题助手

一个基于 Tkinter 的本地刷题工具，适合选择题、判断题和多选题练习。程序会从 `题库` 文件夹读取 Excel 题库，并自动记录每个题库的学习进度。

## 运行

```powershell
uv run python main.py
```

也可以继续使用项目里的 `运行.bat` 启动。

## 题库格式

题库文件放在 `题库` 文件夹中，格式为 `.xlsx`。每一行是一道题，至少包含 3 列，第 4 列可选填解析：

| 第 1 列 | 第 2 列 | 第 3 列 | 第 4 列 |
| --- | --- | --- | --- |
| 题干 | 选项 | 答案 | 解析 |

选择题选项支持 `A. xxx B. xxx`、`A、xxx B、xxx` 或 `A:xxx B:xxx` 这类格式。判断题答案填写 `对` 或 `错`。多选题答案可用逗号或连续字母，例如 `A,C` 或 `AC`。解析列可直接填写内容，也可写成 `解析：内容`。

## 功能

### 题库管理

- 启动后自动扫描 `题库` 文件夹，读取所有 `.xlsx` 题库文件。
- 如果根目录下没有 `题库` 文件夹，程序会自动创建。
- 顶部以下拉框展示题库，选择后即可切换当前题库。
- 每个题库独立保存学习进度，再次打开时会从上次位置继续。
- 支持普通题库和自动生成的错题集文件一起显示和练习。

### 答题练习

- 支持单选题、多选题和判断题。
- 单选题、判断题点击选项即可选中。
- 多选题可连续点击多个选项，再点击“确认提交”。
- 顶部显示当前进度和题目类型，方便掌握练习位置。
- 可开启“自动提交”，单选题和判断题选中后会立即判题。

### 判题与反馈

- 提交后会立即显示正确或错误结果。
- 答对后自动进入下一题，减少重复点击。
- 答错后会显示正确答案，并高亮错误选择和正确选项。
- 可点击“查看解析”显示第 4 列内容；如果题目第 4 列填写了解析，提交后会在选项下方显示“解析：内容”，背题模式会直接显示解析。
- 可点击“修改解析”录入或修改第 4 列解析内容，并写回当前题库 Excel。
- 未选择答案时，提交按钮保持禁用，避免误操作。

### 错题集

- 答错的题目会自动写入对应的错题集文件。
- 可以手动将当前题加入错题集。
- 在错题集模式下，可以把已经掌握的题目移出错题集。
- 写入错题集时会检查重复题目，避免同一道题反复保存。

### 跳转与进度

- 支持输入题号并按回车跳转到指定题目。
- 每次切换题目时自动保存当前题库进度。
- 进度数据保存在 `study_progress.json`，不会修改原题库内容。

### 界面

- 使用本地 Tkinter 界面，无需联网即可运行。
- 顶部题库下拉、功能色块、答题区和底部导航区分明。
- 主要按钮、题库下拉、题干和选项使用统一圆角样式，适合日常反复练习使用。
